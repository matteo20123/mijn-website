Projecten

Raad het getal

Deze code laat de speler een getal tussen 1 en 100 raden binnen 7 pogingen. Het te raden getal is 77. Bij elke poging voert de speler een gok in: is de gok correct, dan verschijnt *"proficiat je hebt het getal geraden."*; is de gok te laag, dan verschijnt "Het getal is groter." en bij een te hoge gok verschijnt "Het getal is kleiner." Lukt het niet binnen 7 pogingen, dan verschijnt "Volgende keer beter."

Geef de juiste pincode

Deze code heeft een vooraf gekozen pincode. enkel de persoon die de pincode kent kan door deze code. Als je 3 keer de foute code ingeeft dan gooit het systeem je eruit. .

Even of oneven

Deze code laat de speler een cijfer kiezen. De speler mag niets anders dan een getal ingeven. Als de speler een getal ingeeft kijkt de code of het getal groter is dan 0. Als dit het geval is dan is het cijfer positief, anders is het cijfer negatief. Nadat de code kijkt of het cijfer positief of negatief is, doet de code het getal delen door 2. Als het getal een kommagetal is geeft het aan dat het cijfer oneven was anders geeft het aan dat het cijfer even was.

medeklinkers

Deze code haalt het woord of de zin van de gekozen speler en haalt alle klinkers eruit. Dan print hij alle medeklinkers zo ziet de speler het woord zonder klinkers.

Pallingdroom

Deze code neemt het gekozen woord van de speler en draait het om. Als het omgedraaide woord hetzelfde is als het normale woord geeft de code aan dat dit een pallingdroom is.

Unieke letters in een woord

Deze code neemt het woord van de speler en kijkt naar de letters en haalt dezelfde letters eruit waardoor je het woord terugkrijgt maar zonder dubbele letters.
